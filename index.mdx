OpenClaude Portable is a zero-install AI coding agent you can carry on a USB drive and run on any Windows, Linux, or macOS machine. It bundles Node.js, the OpenClaude engine, and a web-based chat dashboard — all self-contained inside a single folder. Plug in, launch, and start coding with your choice of AI provider.

## How it works

OpenClaude Portable wraps the open-source [OpenClaude](https://github.com/gitlawb/openclaude) engine in a cross-platform launcher that requires nothing from the host machine. On first run it downloads Node.js (~25 MB) and the OpenClaude engine (~5 MB), then guides you through provider setup. Every subsequent launch skips setup and opens the main menu directly.

<Steps>
  <Step title="Copy the folder to your USB drive (or any directory)">
    Download and extract the OpenClaude Portable release to your USB drive or a local folder. No installer, no admin rights required.
  </Step>
  <Step title="Launch the starter script">
    Run `START.bat` on Windows or `./start.sh` on Linux/macOS. On first run, Node.js and the engine download automatically.
  </Step>
  <Step title="Choose an AI provider">
    Pick from 7 providers — OpenRouter and NVIDIA NIM have free tiers. Enter your API key, which is stored only in `data/ai_settings.env` on your drive.
  </Step>
  <Step title="Start coding">
    Select **Launch AI** from the main menu. The agent runs in your terminal or open the **Web Dashboard** for a browser-based chat interface.
  </Step>
</Steps>
